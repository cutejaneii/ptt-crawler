# ptt-crawler

本專案利用[requests](https://github.com/requests/requests)取得ptt文章

## 使用方法

本專案提供五種模式，請詳見以下說明。

### 取得最新文章

<pre><code>git clone </code></pre>

### 取得「某頁數」到「某頁數」的文章

<pre><code>git clone </code></pre>

### 取「某一頁」的文章

<pre><code>git clone </code></pre>

### 利用「關鍵字」取得文章

<pre><code>git clone </code></pre>
